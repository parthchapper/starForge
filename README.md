## Starforged

**Journey of a Hydrogen Atom**

Starforged is an interactive website that traces the life of a single hydrogen atom from the Big Bang, through the formation of stars, planets, and life, to its role in human consciousness. The website features immersive visuals, ambient music, and a first-person narrative from the atomâ€™s perspective.

## Features

* Four immersive chapters:

  1. Overview
  2. The Big Bang
  3. Laws of Being
  4. Now
* Starry background with smooth, frosted-glass cards
* Ambient music that starts automatically when the user scrolls
* Fully responsive layout with gradient titles and elegant design

## Installation

1. Clone this repository:

```bash
git clone https://github.com/yourusername/starforged.git
```

2. Navigate to the project directory:

```bash
cd starforged
```

3. Install Streamlit if not already installed:

```bash
pip install streamlit
```

## Usage

Run the app using:

```bash
streamlit run starforged.py
```

Scroll down the page to start the ambient music and explore the story.
